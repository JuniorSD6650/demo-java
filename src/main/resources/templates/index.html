Gestor de Hábitos

Sobre el Proyecto

Es un gestor de hábitos, que permite gestionar usuarios, hábitos y sus registros. Implementa relaciones uno a muchos y muchos a muchos, aplicando principios de programación orientada a objetos y arquitectura MVC.

Descripción del Proyecto

Realizar un programa como caso práctico de lo aprendido en el curso, empleando el paradigma de programación orientado a objetos, usando el lenguaje de programación Java y el framework Spring.

📌 Requisitos:
  • Relación uno a muchos: El programa deberá emplear una base de datos en MySQL para gestionar los datos con Java y Spring Boot, aplicando una relación de uno a muchos entre las tablas.
  • Relación muchos a muchos: Se debe aplicar una relación de muchos a muchos entre las entidades correspondientes.
  • Implementación de la organización de paquetes en el proyecto, incluyendo entidades, servicios, repositorios, controladores y vistas.
  • Se deben incluir las funcionalidades de registrar, buscar, listar, editar y eliminar en la aplicación.
